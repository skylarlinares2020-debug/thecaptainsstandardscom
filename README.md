# Captain's Standard LLC

A modern, mobile-responsive marketing website for Captain's Standard LLC — a mobile boat care and detailing business serving East Tennessee.

## Tech Stack

- **Framework**: TanStack Start (React, SSR)
- **Styling**: Tailwind CSS v4 + custom CSS variables
- **Forms**: Netlify Forms (serverless form handling)
- **Icons**: Lucide React
- **Fonts**: Inter (sans) + Playfair Display (serif) via Google Fonts
- **Hosting**: Netlify

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at http://localhost:3000. For full Netlify feature emulation (forms, etc.):

```bash
/opt/buildhome/node-deps/node_modules/.bin/netlify dev --port 8889
```
